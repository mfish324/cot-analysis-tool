# COT Analysis Charts

This directory contains charts generated by the COT analysis tool.

## Chart Types

### 1. Comprehensive Chart (`*_comprehensive.png`)
Three-panel chart showing all key metrics:
- **Panel 1: COT Index** - 0-100 scale with overbought (>80) and oversold (<20) zones
- **Panel 2: Net Positions** - Speculators vs Commercials net positioning
- **Panel 3: Open Interest** - Total market volume over time

### 2. COT Index Chart (`*_cot_index.png`)
Shows the COT Index over time with:
- Current index value highlighted
- Overbought zone (80-100) shaded in red
- Oversold zone (0-20) shaded in green
- Neutral line at 50

**Interpretation:**
- Index > 80: Speculators at maximum bullish positioning (contrarian bearish signal)
- Index < 20: Speculators at maximum bearish positioning (contrarian bullish signal)
- Index ~50: Neutral positioning

### 3. Net Positions Chart (`*_net_positions.png`)
Two-panel chart showing:
- **Top Panel**: Net positions for Speculators (red) and Commercials (green)
- **Bottom Panel**: Open Interest (market volume)

**Interpretation:**
- Rising speculator net positions = Increasing bullish bets
- Falling commercial net positions = Hedgers reducing long exposure
- Divergence between specs and commercials often signals market extremes

## Usage

Generate charts with:
```bash
python analyze.py GOLD --chart
python analyze.py "CRUDE OIL" --chart
```

Charts are saved as PNG files with 150 DPI resolution, suitable for reports and presentations.
